# Closet Canvas

Closet Canvas is a recommendation system designed to help users choose outfits based on fashion trends, personal preferences, and fit. This system leverages computer vision models to analyze clothing items and provide personalized outfit suggestions.

The prototype version can be accessed at [Prototype](https://github.com/ajaydsudhir/Closet_Canvas/tree/prototype).
This project was designed with the collaboration of 5 members as part of a semester-long course in Computer Vision as well as a team at Charlotte AI Research. 
The main contributors to this project are Ajay D. Sudhir, Bruh Lemma Yadecha, Abishek Arulkumar Padmapriya, Nitin Chandrasekhar, and Mitchell Nguyen.
